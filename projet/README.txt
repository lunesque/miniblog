MINI PROJET EN DEVELOPPEMENT BACK
Realise par Thuy Hang NGUYEN, MMI2 2024-2025
URL : http://tnguyen.projetsmmichamps.fr/miniprojet

Develope en HTML, CSS, PHP avec l'utilisation de Bootstrap pour la mise en forme
Avec l'application du modele MVC
Base de donnees stockee sur le serveur MySQL a distance de l'hebergeur Alwaysdata

Utilisateur normal : 
    - login: tutu
    - mdp: tutu
Administrateur/proprietaire : 
    - login: toto
    - mdp: toto

Fonctionnalites developpees : 
- Pour internautes non logues : 
    + Creer un compte avec login et mdp
    + Consulter tous les posts et ses commentaires
- Pour utilisateurs logues : 
    + Se connencter et deconnecter
    + Ajouter et modifier sa photo profile
    + Publier des commentaires sur des posts et les modifier, supprimer
- Pour proprietaire () : 
    + Creer, modifier, supprimer des posts
    + Modifier, supprimer des utilisateurs
    + Supprimer des commentaires

Fonctionnalites qui peuvent eventuellement etre developpees : 
- Pour internautes non logues : 
    + Reinitialisation du mot de passe
- Pour utilisateurs logues : 
    + Modifier login, mdp, photo profil
    + Repondre aux commentaires
